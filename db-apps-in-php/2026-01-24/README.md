## Setting up the MAMP PHP/MySql Environment on Windows

See https://www.wa4e.com/software-mamp.php. This course recommends installing MAMP (macOS, Apache, MySQL, PHP). This can all be achieved instead using Dockerfiles and Rancher Desktop / Docker Desktop, so I've done that. See [Dockerfile](./Dockerfile) and [docker-compose.yaml](./docker-compose.yml).

Just run:
```
docker-compose up
```
